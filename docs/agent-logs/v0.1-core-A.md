## 关键设计决策

- **events 三段式**：readLoop → events(cap 64) → fanoutLoop → broadcast 各 sub
- **tryPublish 非阻塞** + **broadcast 非阻塞**：慢 sub 不阻塞 readLoop
- **输入背压**：inputCh cap 64 + ErrInputFull
- **资源清理**：closeOnce + connMu 写锁置 nil
- **状态同步**：atomic.Int32 + atomic.Pointer[Info]
- **避免循环依赖**：`connect/auth_convert.go` 是 free function（接收原始字段）
