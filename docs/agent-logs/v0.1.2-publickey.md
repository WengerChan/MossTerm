# v0.1.2 · 后端工程师 task（publickey auth）

**派活时间**：2026-07-13  
**目标**：接通 SSH publickey 认证

## 任务描述

- `connect.Deps` 加 `Secrets secret.Store` 字段
- `connect.PublicKeyAuth` 加 `KeyID` 字段（Signer 已解析 / KeyID 延迟加载）
- `sshclient.Connector` 加 `secrets` + `loadSigner` 真实实现
- `authMethods` 改为 `*Connector` 方法（能用 c.loadSigner）
- `session.MemoryManager.WithSecrets` 注入器
- `cmd/mossterm/main.go` 链式 `WithSecrets(sec)` 注入

## 关键设计决策

- **依赖注入**（不是直接 import）：保留 connect 包协议无关
- **publickey 解析放在 sshclient 层**：connect 只做形态转换
- **PublicKeyAuth 双模式**：Signer 已解析 / KeyID 延迟加载
- **signerCache LRU**：per-connector 缓存已解析 ssh.Signer（不存明文私钥）

## 数据流

```
wailsbindings → app → session.MemoryManager
  .WithSecrets(sec) 注入
  → Open 构造 connect.Deps{Secrets: sec}
  → sshclient.Connector.secrets
  → c.authMethods(PublicKeyAuth{KeyID})
  → c.loadSigner(keyID) → secrets.Get → bytes → Signer
```
