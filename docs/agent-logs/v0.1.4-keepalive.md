## 关键设计决策

- **失败处理：不主动关 client**，只 log + 退出协程（避免影响其他 session）
- **SendRequest 超时**：x/crypto v0.22.0 的 ssh.Client.SendRequest 走嵌入 Conn interface（3 返回值无 ctx），用 goroutine + select + time.NewTimer 模拟
- **done per-Connector**（不是 per-Dial）：单次 Close 关所有 keepalive
- **Close 不在 connect.Connector 接口里**：keepalive 资源由 sshclient 自治
