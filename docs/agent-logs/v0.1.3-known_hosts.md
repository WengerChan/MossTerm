## 关键设计决策

- **自实现 vs 标准库**：`x/crypto/ssh/knownhosts` 不导出 DB 类型，无法做 first-use trust + 自动持久化
- **3 级 host key 优先级**：KnownHosts > HostKeyCb > InsecureIgnoreHostKey 兜底
- **智能 callback 策略**：匹配放行 / 不匹配拒绝（MITM）/ 未找到自动 Add
- **路径策略**：`~/.config/mossterm/known_hosts`（与 OpenSSH `~/.ssh/known_hosts` 隔离）
- **v0.1.2 修 bug 时误删了"创建空文件"代码**——本次重写补回 + 加测试
