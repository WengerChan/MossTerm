# v0.5.11 · x/crypto v0.22 → v0.31 升级

> **最小变更升级**：v0.22 → v0.31.0 锁紧上游稳定段，零代码改动，全套测试通过。
> 早前 v0.5.1 时代锁 v0.22.0 是为了规避 v0.33+ 的"argon2 类型移除 + ssh.NewClient 拆两步"破坏性变更。
> v0.31 是最后一个保留旧 API 的稳定版本 —— 升到 v0.31 拿上游安全补丁，不引入 v0.33+ 的破坏性变更。
